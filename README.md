**Dolibarr 23.0.2** – La solution ERP/CRM open source complète, simple et modulable pour TPE, PME et associations.

### Points forts de cette version
- ERP/CRM tout-en-un : facturation, stocks, comptabilité, projets, RH, etc.
- Architecture modulaire très flexible
- Interface moderne et responsive
- Facile à installer et à utiliser
- Multi-sociétés et multi-utilisateurs
- Très utilisé par les petites et moyennes entreprises
